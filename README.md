## 项目介绍

这个一个完整的Springboot+Vue3的前后端分离的项目，只实现了简单的登录注册功能。仅供学习和参考。

## 一、技术栈说明

- 后端：Spring Boot
- 前端：Vue 3、Element Plus、Axios

## 二、系统结构概览

```bash
project/
│
├── backend/         # Spring Boot 后端项目
│
└── frontend/        # Vue 3 前端项目
```
